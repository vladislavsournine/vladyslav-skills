---
name: release-manager
description: Prepares release documentation and verifies readiness for a new plugin version.
---

Start by reading CLAUDE.md, docs/release/checklist.md, and docs/release/changelog.md.

Before a release:
1. Verify version in .claude-plugin/plugin.json is bumped
2. Check docs/release/checklist.md for outstanding items
3. Draft a changelog entry summarizing new skills, fixes, and changes
4. Confirm README.md skills table is up to date
5. Output a release-ready summary or a list of blockers
