# Stack: Swift / iOS (SwiftUI)

This fragment is composed into the `init-project` subagent prompt when the user selects Swift as a frontend/mobile stack.

## Pre-flight inputs (already collected)

- `bundle_id_prefix` — e.g. `com.vlad`
- `deployment_target` — e.g. `17.0` (default if not provided)

## Directories

Create:

```
swift/                  # parent grouping (only if other mobile stacks coexist; otherwise omit and place app/, tests/ at root)
app/
app/Resources/
tests/
```

## .gitignore additions

Append to `.gitignore`:

```
DerivedData/
*.xcuserstate
*.xcworkspace/xcuserdata/
*.xcodeproj/
Pods/
```

`.xcodeproj/` is gitignored because it is generated by xcodegen from `project.yml`. Only `project.yml` is committed.

## Install xcodegen if missing

```bash
which xcodegen || brew install xcodegen
```

If `brew` itself is missing, return `status: error`: `"Homebrew not found — Swift project bootstrap requires Homebrew. Install from https://brew.sh and re-run."`

## Files

### `project.yml`

Read `<plugin>/skills/init-project/assets/swift/project.yml`. Write to `project.yml` at the project root, replacing:

- `PROJECT_NAME` → actual project name
- `BUNDLE_ID_PREFIX` → bundle ID prefix from pre-flight (e.g. `com.vlad`)
- `PROJECT_NAME_LOWER` → lowercased project name
- `DEPLOYMENT_TARGET` → deployment target from pre-flight (default `17.0`)

If `assets/swift/project.yml` cannot be located, return `status: error`: `"Cannot find skills/init-project/assets/swift/project.yml in vladyslav-skills plugin. Please reinstall or run git pull."`

### `app/Info.plist`

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>UILaunchScreen</key>
    <dict/>
</dict>
</plist>
```

### `app/<ProjectName>App.swift`

Replace `<ProjectName>` with the actual project name.

```swift
import SwiftUI

@main
struct <ProjectName>App: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

### `app/ContentView.swift`

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hello, World!")
    }
}
```

### `.claude/settings.local.json`

```json
{
  "permissions": {
    "allow": ["Bash(xcodegen:*)", "Bash(xcodebuild:*)"]
  }
}
```

## Generate the project

```bash
xcodegen generate
```

Confirm `<ProjectName>.xcodeproj` was created. If `xcodegen generate` fails, return `status: error` with the captured stderr.
